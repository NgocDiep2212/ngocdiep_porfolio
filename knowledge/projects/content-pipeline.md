## Overview

AI Content Pipeline with 5 API integrations. Stack: n8n, Zalo OA, Leonardo AI, TikTok API, Human-in-the-Loop.

4-stage automation: webhook trigger → AI media generation → human approval gate → TikTok auto-publish. Zero manual publish steps.

## What was built

- Integrated 5 third-party APIs end-to-end: each with OAuth/token auth, payload mapping, and conditional error branching.
- Pipeline pauses at human approval gate — operator receives a bot notification and must confirm before publish fires.

## Impact

- End-to-end under 3 minutes; 5 APIs integrated; 0 manual publish steps.
